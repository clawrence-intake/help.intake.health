# Project: Intake Health Help Site

## Goal
Build a blazingly fast, lightweight static knowledge base / help desk website for Intake Health.

## Requirements
- Static site (no server-side rendering, no heavy frameworks)
- Hosted on GitHub Pages via `clawrence-intake` account
- Must match Intake Health branding from https://www.intake.health
- Content sourced from the support docs in `content/Customer Support/`
- Live and publicly accessible when done

## Branding Reference
- Main site: https://www.intake.health
- Colors: Dark/navy backgrounds, green/teal accents, white text
- Clean, modern, professional sports/health aesthetic
- Logo and assets: Pull from main website

## Content Structure (from support docs)
1. Product Overview — What InFlow is, how it works, LED states
2. Setup & Installation — Mounting, actiongraphic, custodial setup
3. Charging & Maintenance — Battery charging, humidity maintenance, cleaning
4. Troubleshooting & FAQ — Common issues, diagnostic steps
5. Returns & Lease Program — Return kits, lease details, RMA
6. Pricing & Purchasing — Purchase vs lease, volume discounts
